Writer

api:
- FileAppender: append new info in file                 
- FileOverwriter: overwrite existent file
- FileSaver: save modified file
- FileUpdater: gestisce la modifica di un file già esistente
    ✔ leggere il file
    ✔ parsare il contenuto
    ✔ modificare solo la parte necessaria
    ✔ serializzare di nuovo
    ✔ riscrivere il file aggiornato
